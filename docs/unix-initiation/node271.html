Macros

Une séquence de traitements "emacs" peut être mémorisée afin d'être rejouée une ou plusieurs fois ultérieurement. Cette séquence peut être nommée, sauvegardée et restaurée au cours d'une autre session "emacs".

- (
Définition de la macro
liste des traitements emacs
- )

Séquence de touche Description
- Execution de la dernière macro créée.
- n - Execution n fois de la dernière macro créée.
- name-last-kbd-macro Association d'un nom à la macro dernièrement définie.

Pour pouvoir réutiliser cette macro au cours d'une autre session, on peut :

  • Soit sauver sa définition dans un fichier externe qu'il faudra recharger avant utilisation (commande " - load-file").
  • Soit sauver sa définition dans le fichier d'initialisation ".emacs". Il suffit de l'ouvrir, de s'y positionner, et d'entrer la commande :
    - insert-kbd-macro  nom_de_macro 
    Lors d'une prochaine session "emacs", la macro sera executée par la commande - nom_de_macro.

Remarque 19..11 :


Pour annuler une modification de texte, il existe la commande "undo" ( représentée par - ). Cette modification de texte est liée à un buffer. Elle peut être répétée plusieurs fois (manuellement ou par la sequence - n - ). Les modifications de texte sont mémorisables dans la limite de 8000 caractères par buffer.
