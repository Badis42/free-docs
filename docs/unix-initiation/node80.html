Résumé

Symbole Type Action Exception
       
$ Caractère spécial Substitue la valeur d'une variable. N.A.
>, >> Caractère spécial Redirige la sortie standard vers un fichier. N.A.
< Caractère spécial Redirige l'entrée standard à partir d'un fichier. N.A.
| Caractère spécial Combine plusieurs commandes dans un pipe. N.A.
Caractère spécial Délimiteur dans une commande. N.A.
? Métacaractère Masque un caractère, sauf le point en première position. N.A.
[] Métacaractère Définit une classe de caractères à masquer. N.A.
* Métacaractère Masque toutes chaînes de caractères, sauf le point en première position. N.A.
\ (back slash) Caractère d'échappement Annule l'interprétation du caractère suivant. Aucune.
' (simple quote) Caractère d'échappement Annule l'interprétation de tous les caractères entre les « ' ». Aucune.
" (double quotes) Caractère d'échappement Annule l'interprétation de tous les caractères entre les « " ». \ (back slash), « $ » (symbole dollar), « " » (double quotes), « ` » (back quote).
