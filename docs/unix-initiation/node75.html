Utilisation des métacaractères « [] »

Les crochets ouvrants et fermants []« [] » définissent une classe de caractères, dont un et un seul sera masqué.

Exemple 4..7 :


echo [abc]??  Tous les fichiers dont le nom fait 3 caractères et dont la première lettre est soit « a », soit « b », soit « c ».
echo ?[a-zA-Z]  Tous les fichiers dont le nom fait 2 caractères et dont le dernier caractère est une lettre (majuscule ou minuscule).
echo [!a-zA-Z]??  Tous les fichiers dont le nom fait 3 caractères et dont le premier n'est pas une lettre.
