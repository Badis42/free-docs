Boucle du type "for"

C Shell Bourne Shell
   
Syntaxe
foreach variable (liste) for variable in liste
commandes do
end commandes
  done
Exemple
set nom="part" nom="part"
foreach fichier (`ls $nom.?a`) for fichier in `ls $nom.?a`
pr $fichier | lp - do
end pr $fichier | lp -
  done
