Commandes pwd et cd

pwd  = Print Working Directory
cd  = Change Directory

Syntaxe :


pwd Affiche le chemin d'accès absolu du répertoire courant.
cd repertoire Change le répertoire courrant.

Exemple 2..1 : 


% pwd
/home/users/bart
% cd /home/users/schmoll
