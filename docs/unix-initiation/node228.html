Boucle du type "while"

C Shell Bourne Shell
   
Syntaxe
while (expression) while [ expression ]
commandes do
end commandes
  done
  Remarque
  Ne pas oublier les espaces pour séparer "expression" des crochets ("[]").
Exemple
while ($#argv) while [ $# -gt 0 ]
echo $argv[1] do
shift echo $1
end shift
  done
